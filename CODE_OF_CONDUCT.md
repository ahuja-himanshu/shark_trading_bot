# Contributor Covenant Code of Conduct

We pledge to make participation in this project a harassment-free experience for everyone, regardless of identity, background, experience, or viewpoint.

Expected behaviour includes empathy, respectful technical disagreement, constructive feedback, and responsible handling of security reports. Unacceptable behaviour includes harassment, threats, discriminatory language, publishing another person's private information, or using project access to target accounts or funds.

Project maintainers may remove, edit, or reject contributions and ban participants whose behaviour is inappropriate, threatening, or harmful. Report conduct concerns privately to the repository owner through their verified GitHub profile. Maintainers will respect the reporter's privacy as far as reasonably possible.

This policy is adapted from Contributor Covenant 2.1: <https://www.contributor-covenant.org/version/2/1/code_of_conduct.html>.
